# 🎮 Tic Tac Toe Web Application

A modern, interactive **Tic Tac Toe game** built using HTML, CSS, and JavaScript.  
This project includes both **Player vs Player** and **Player vs Computer (AI)** modes with smooth UI, animations, and score tracking.

## 🧠 Game Logic

- Game board is stored in an array of 9 elements
- Win conditions are checked using predefined winning combinations
- AI uses **Minimax Algorithm** to ensure optimal moves
- Score updates dynamically after each match

---

## 🎯 How to Play

1. Open the game in browser
2. Choose mode:
   - Player vs Player
   - Player vs Computer
3. Click on any box to place your mark (X or O)
4. First to align 3 marks wins!
5. Click Restart to play again
