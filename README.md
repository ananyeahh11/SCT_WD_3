🧩 Project Title:

Tic-Tac-Toe Web Application (Player vs Player / Player vs Computer)

📌 Objective

Build an interactive Tic-Tac-Toe web application that allows:

Two players to play against each other, OR
A single player to play against the computer

The application must:

Handle user clicks
Track game state
Check winning conditions
Provide an engaging UI experience
🚀 Features Implemented
🎮 Gameplay
Player vs Player mode (X vs O)
Player vs Computer mode
Turn-based gameplay system
🧠 Game Logic
Real-time board state tracking
Win condition detection (rows, columns, diagonals)
Draw detection
Game reset functionality
🤖 AI Mode
Computer opponent uses Minimax Algorithm
Computer plays optimally (hard/impossible mode)
🎨 User Interface
Clean modern dark-themed UI
Responsive 3x3 game grid
Hover and click interactions
Winning cell highlight animation
🔄 Game Controls
Restart game button
Mode selection (PvP / CPU)
🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla)
DOM Manipulation
Minimax Algorithm (AI logic)
📁 Project Structure
tic-tac-toe/
│── index.html

│── README.md
⚙️ How It Works
Player selects game mode (PvP or CPU)
Game board is initialized
Players take turns placing X and O
After every move:
Game checks for a winner
Or checks for a draw
In CPU mode:
AI calculates best possible move using Minimax algorithm
Game ends when win/draw condition is met
